This repository is for bug reports, feature requests, and feedback for **CUStats**.

## About CUStats

CUStats lives in your macOS menubar and shows your Claude AI usage in real-time:

- **Real-time Usage Tracking** - Monitor 5-hour, 7-day, and model-specific limits
- **Live Countdown Timer** - See exactly when your limits reset
- **Historical Views** - Track usage patterns by day, week, or month
- **Multi-Account Support** - Manage multiple Claude accounts
- **Native macOS App** - Built with SwiftUI, lightweight and private

## How to Report Issues

### Bug Report
Found a bug? [Create a bug report](../../issues/new?template=bug_report.yml) with:
- Your app and macOS version
- Steps to reproduce the issue
- What you expected vs what happened

### Feature Request
Have an idea? [Request a feature](../../issues/new?template=feature_request.yml) with:
- The problem you're trying to solve
- Your proposed solution

### General Feedback
Want to share thoughts? [Submit feedback](../../issues/new?template=feedback.yml)

## Before Submitting

1. **Search existing issues** to avoid duplicates
2. **Use the appropriate template** for your submission
3. **Be specific** - include version numbers and steps to reproduce
